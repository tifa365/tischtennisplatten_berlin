# Berlin Ping Pong Tables

A collection of ping pong table locations in Berlin.

## Data

The dataset contains 2,710 ping pong table locations across Berlin with their geographic coordinates.

### Format

The data is provided in JSON format with the following structure:

```json
[
  {
    "lat": 52.4907,
    "lon": 13.4726
  },
  ...
]
```

Each entry contains:
- `lat`: Latitude coordinate
- `lon`: Longitude coordinate

## File

- `berlin_pingpong_tables.json` - Complete dataset of all ping pong table locations
